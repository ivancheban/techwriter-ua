# Як опублікувати

Ми приймаємо оригінальні статті про технічне письмо, UX-райтинг, документування API та супутні інструменти. Поділіться своїм досвідом з українською спільнотою!

### Як подати заявку

Будь ласка, надішліть свою пропозицію на електронну пошту `writers@uawriters.space`.

Вкажіть:
1. **Ваше ім'я**
2. **Тему статті**
3. **Короткий опис або посилання на чернетку**

Ми зв'яжемося з вами найближчим часом!
